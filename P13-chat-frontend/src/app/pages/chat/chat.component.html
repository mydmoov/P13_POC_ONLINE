Chatbot History

Cette application est un chatbot interactif développé en utilisant Angular pour le frontend et Spring Boot avec Spring Data JPA pour le backend. Elle permet aux utilisateurs de communiquer en temps réel grâce à l'intégration de WebSocket.

Frontend

Le frontend de l'application est construit avec Angular. Il offre une interface utilisateur intuitive et réactive, permettant aux utilisateurs de saisir et d'envoyer des messages. Les messages sont affichés en temps réel grâce à l'utilisation de WebSocket, qui permet une communication bidirectionnelle entre le client et le serveur.

Backend

Le backend est développé en utilisant Spring Boot, un framework Java qui facilite la création d'applications robustes et évolutives. Spring Data JPA est utilisé pour la gestion des données, permettant une interaction facile avec la base de données. Les messages envoyés par les utilisateurs sont stockés dans une base de données relationnelle, assurant la persistance des données.

WebSocket

L'intégration de WebSocket permet une communication en temps réel entre le client et le serveur. Lorsqu'un utilisateur envoie un message, celui-ci est transmis au serveur via WebSocket, puis diffusé à tous les clients connectés. Cela garantit que tous les utilisateurs voient les nouveaux messages instantanément, sans avoir besoin de recharger la page.
